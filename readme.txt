## API

### Генерация и получение кода авторизации

**URL**: `/api/generate_code/`

**Метод**: `POST`

**Параметры запроса**:

- `phone_number`: Номер телефона пользователя

**Пример ответа**:

```json
{
    "status": "success",
    "message": "Authorization code is abcd"
}

### Активация инвайт кода

**URL**: `/api/referral_code/`

**Метод**: `POST`

**Параметры запроса**:

- `phone_number`: Номер телефона пользователя
- `referral_code`: Код приглашения другого пользователя

**Пример ответа**:

```json
{
    "status": "success"
}

### Получение информации о профиле

**URL**: `/api/authorize/`

**Метод**: `POST`

**Параметры запроса**:

- `phone_number`: Номер телефона пользователя
- `authorization_code`: Код для авторизации


**Пример ответа**:

```json
{
    "status": "success",
    "user": {
        "phone_number": "11111111",
        "invite_code": "f58e22",
        "referral_code": "55edc2",
        "used_count": 1,
        "active": true,
        "referrals": [
            "9612282405"
        ]
    }
}
